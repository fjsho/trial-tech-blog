# Product Overview

## プロダクト概要

Tech Blog は、エンジニア個人の学びと成長を記録・共有するための技術ブログサイトです。プログラミング、ガジェット、読書という3つの軸で構成されたパーソナルポートフォリオサイトとして機能します。

## Core Features

- **ブログ管理**: 技術メモ、スキルアップの記録、外部プラットフォーム(Zenn/Qiita)の記事管理
- **ガジェット紹介**: 愛用しているガジェットやツールの紹介・レビュー
- **読書記録**: 読んだ技術書の記録とレビュー管理
- **外部連携**: GitHub、Twitter(X)などのソーシャルリンク
- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応

## Target Use Case

### メインユースケース
- エンジニアとしての学習記録を一元管理
- 技術的な知見や経験を継続的に発信
- ガジェットや技術書のレビューを蓄積
- ポートフォリオとしての自己表現の場

### ターゲットユーザー
- 学習記録を残したいエンジニア
- 技術発信を行いたい開発者
- ガジェットや技術書に興味があるテック愛好家
- 自分の成長過程を可視化したい人

## Key Value Proposition

### 主な価値
- **統合管理**: ブログ、ガジェット、読書という3つのカテゴリを1つのサイトで管理
- **外部連携**: Zenn/Qiita等の外部プラットフォームの記事も統合表示可能
- **シンプルな構成**: 使いやすく、メンテナンスしやすいシンプルな設計
- **モダンなUI**: Astroベースの高速で美しいユーザーインターフェース

### 差別化ポイント
- Supabaseを使用した効率的なデータ管理
- 静的サイト生成による高速なパフォーマンス
- カテゴリごとに異なるカラーテーマでの視覚的な区別
- 日本語フォント(Noto Sans JP)による読みやすさ
