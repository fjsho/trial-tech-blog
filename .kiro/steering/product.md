# Product Overview

## プロダクト概要

**くじらのTech** は、エンジニア個人の学びと成長を記録・共有するための技術ブログサイトです。「プログラミングのこと、好きなガジェット、読書の記録」をテーマに、3つの軸で構成されたパーソナルポートフォリオサイトとして機能します。

## Core Features

- **ブログ管理**: 外部プラットフォーム(Zenn/Qiita)の記事をAPI経由で取得・統合表示
- **ガジェット紹介**: Markdownファイルで管理されたガジェットやツールの紹介・レビュー
- **読書記録**: Markdownファイルで管理された技術書の記録とレビュー
- **カテゴリフィルタ**: ガジェットページでのカテゴリ別フィルタリング機能
- **外部連携**: GitHub、Twitter(X)などのソーシャルリンク
- **レスポンシブデザイン**: モバイル、タブレット、デスクトップに対応

## Target Use Case

### メインユースケース
- エンジニアとしての学習記録を一元管理
- 技術的な知見や経験を継続的に発信
- ガジェットや技術書のレビューを蓄積
- ポートフォリオとしての自己表現の場

### ターゲットユーザー
- 学習記録を残したいエンジニア
- 技術発信を行いたい開発者
- ガジェットや技術書に興味があるテック愛好家
- 自分の成長過程を可視化したい人
- きれいに情報が整理されたサイトで学びを集約したい人

## Key Value Proposition

### 主な価値
- **統合管理**: ブログ、ガジェット、読書という3つのカテゴリを1つのサイトで管理
- **外部連携**: Zenn/Qiita等の外部プラットフォームの記事も統合表示可能
- **シンプルな構成**: 使いやすく、メンテナンスしやすいシンプルな設計
- **モダンなUI**: Astroベースの高速で美しいユーザーインターフェース

### 差別化ポイント
- 外部プラットフォーム（Zenn/Qiita）のAPI統合による自動記事取得
- Astro Content Collectionsによるガジェット・書籍の管理
- 静的サイト生成による高速なパフォーマンス
- カテゴリごとに異なるカラーテーマでの視覚的な区別
- 日本語フォント(Noto Sans JP)による読みやすさ

## 収益化

### Amazonアソシエイト
- サイト内のガジェット・書籍紹介にはAmazonアソシエイトのアフィリエイトリンクが含まれる
- ユーザーに対して透明性を保つため、Aboutページでアフィリエイトリンクの使用を明示
- 購入を促すのではなく、実際に使用した製品や読んだ書籍の情報共有が主目的
